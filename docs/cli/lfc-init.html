3.1. lfc-init: Initialize LFC repo¶

This creates two folders (if they don’t exist):

  • .lfc/

  • .lfc/cache/

And several files:

  • .lfc/config

  • .lfc/.gitignore

Usage:
$ lfc init [OPTIONS]
Options:
-h, --help

Display this help message and exit
