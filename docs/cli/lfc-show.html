3.8. lfc-show: Print contents of a large file to STDOUTÂ¶

Print contents of a large file to STDOUT, even in a bare repo. This function does not decode the bytes so that binary files can be piped from bare repos through STDOUT.

Usage:
$ lfc show FNAME [OPTIONS]
Inputs:

  • FNAME: name of large file (including .lfc) to show

Options:
-h, --help

Display this help message and exit

--ref REF

Use specified git ref, for example commit hash or branch name; default is HEAD
