3.9. lfc-ls-files: List large filesÂ¶

This command lists all .lfc files matching specified constraints. If no arguments are given, it will list all .lfc files in the current working directory or any folder within (recursively). Users can limit this to all files starting with m, for example, or apply any other constratins.

Usage:
$ lfc ls-files [PAT1 [PAT2 ...]] [OPTIONS]
Inputs:

  • PAT1: First pattern for files to list

  • PAT2: Second pattern for files to list

Options:
-h, --help

Display this help message and exit
