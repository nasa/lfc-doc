lfc: A git extension to track large files¶

Large File Control (lfc) is a Python package to track large files in git repositories. It provides both an API (see LFCRepo) and a command-line interface (see lfc.cli).

The package works by calculating SHA-256 hashes of each “large” file. Users determine which files are “large” by running lfc_add() on them. The package can also share these large files remotely using a simultaneous SSH and SFTP connection to a remote server.

  • cli: Command-line interface to lfc
  • lfcrepo: Interface to git repos with large-file control
  • lfcclone: Command-line interface for lfc-clone
  • lfcerror: Errors for lfc modules
