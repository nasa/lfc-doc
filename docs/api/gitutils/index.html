gitutils: Python interface to git repositoriesÂ¶

This package provides the class GitRepo. Instances of this class can run common git commands on the repository from which they were initiated and perform other git-related actions such as modifying the configuration.

To instantiate a repo in the current folder, simply run

from gitutils import GitRepo

repo = GitRepo()
